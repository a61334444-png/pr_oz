# Анализ данных средствами Python

## Описание проекта

Проект разработан в рамках учебной практики по дисциплине «Анализ данных».

Цель работы – получение навыков обработки, анализа и визуализации данных с использованием языка программирования Python и специализированных библиотек.

## Используемые технологии

- Python 3.x
- NumPy
- Pandas
- Matplotlib

## Функциональные возможности

Программа выполняет следующие операции:

1. Генерация набора данных из 1000 случайных чисел.
2. Формирование объекта Series библиотеки Pandas.
3. Очистка данных от цифрового мусора.
4. Расчет статистических характеристик:
   - минимального значения;
   - максимального значения;
   - суммы значений;
   - среднего значения;
   - медианы;
   - стандартного отклонения;
   - количества повторяющихся значений.
5. Формирование объекта DataFrame.
6. Визуализация результатов анализа:
   - линейный график;
   - гистограмма распределения данных;
   - график сравнения отсортированных данных.
7. Сохранение результатов в файл CSV.
